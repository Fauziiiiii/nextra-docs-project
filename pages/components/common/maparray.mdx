# Map Array

Komponen Map Array dalam Total.js Flow digunakan untuk memetakan elemen-elemen dalam array berdasarkan logika atau aturan yang ditentukan. 
Ini memungkinkan transformasi data dalam array secara otomatis, di mana setiap elemen dalam array dapat diubah menjadi bentuk yang berbeda sesuai dengan kebutuhan.

### Contoh Penggunaan

#### 1. Atur Configuration Trigger

```js copy
[
    { sensor: { temperature: 25, humidity: 50 }},
    { sensor: { temperature: 24, humidity: 55 }},
    { sensor: { temperature: 23, humidity: 60 }}
]
```
